git config --global user.name ""

Komenda pozwala ustawić nazwę użytkownika Git. --global sprawia że Git użyje tych informacji do wszystkich rzeczy wykonywanych na systemie

git diff --staged

Pokazuje dokładne zmiany pomiedzy twoim indexem, a ostatnim commitem. Komenda służy do sprawdzenia, jakie zmiany zajdą w repozytorium, jeżeli użyjemy komendy git commit.
